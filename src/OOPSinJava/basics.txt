 🔴 Constructor :- A constructor is a special method in a class that is automatically called when an object is created.
    (Special method that runs automatically when object is created.)
    💡 Key Points:
    The constructor has the same name as the class.
    It doesn’t have a return type (not even void).
    It runs automatically when you create an object using new.
